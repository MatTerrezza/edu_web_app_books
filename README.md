# BookStore - Интернет-магазин книг

Интерактивный интернет-магазин книг с современным дизайном, анимированными карточками товаров и эффектами пузырьков. Проект создан на React с использованием React Router для навигации.

[Ссылка на демо-версию]

## ✨ Особенности

- 🎨 **Современный дизайн** с темной темой и фиолетовыми акцентами
- 🫧 **Анимированный фон** с пузырьками (BubbleBackground)
- 🔄 **Flip-карточки** для отображения товаров
- 🎯 **Кастомные кнопки** с эффектом "blob" (пузырьковый эффект)
- 📱 **Адаптивная верстка** для мобильных устройств
- 🧭 **Маршрутизация** с React Router
- 🏷️ **Фильтрация** книг по жанрам
- 📖 **Детальная страница** товара с описанием

## 🛠️ Технологии

- React 18
- React Router DOM 6
- CSS3 (с кастомными анимациями)
- SVG-фильтры для эффектов

## 📋 Предварительные требования

- Node.js (версия 14 или выше)
- npm или yarn
